# firstflaskapp

Before view:
* Make sure you have python and flask installed: http://www1.cmc.edu/pages/faculty/alee/cs40/penv/installFlaskOnMac.html

To view:

* Clone repo to Desktop
* 'cd' into repo
* Type 'python app.py'
* Open Chrome browser and go to localhost:5000
